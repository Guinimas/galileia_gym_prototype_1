# Galiléia Gym Website
Galiléia Gym is a Communinity Gym accessible to everyone for free. That is project Website development to improve the visibility and management of the Gym, making all the members to be aproached to the purpose's Gym. Health accessible to everyone.

## Features
- System of contributions
- Login of users and contributors
- Gym's Gallery
- Gym's related News and Important public news
- Important events
- Ranking of contributors
- Budget Gym transparency

## Technologies used
- JavaScript
- HTML5
- CSS3
